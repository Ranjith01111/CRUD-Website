# SwiftTicket – Event Ticket Booking App

A full-stack ticket booking web app built with React, TypeScript, and Firebase. Manage your event bookings, discover local events in Coimbatore, and book tickets with ease.

## Features

- **Ticket Management** – Create, edit, cancel, and delete event tickets
- **Live Recommendations** – Browse upcoming events in Coimbatore
- **Nearby Theaters** – Find movie theaters around Coimbatore
- **E-Pass** – View a digital pass with QR code for each booking
- **Chat Support** – Built-in assistant to help with your bookings
- **Authentication** – Secure sign-in backed by Firebase Auth

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set your credentials in `.env.local` (see `.env.example`)
3. Run the app:
   `npm run dev`
